Courses Offered

The Computer Science Department offers a comprehensive curriculum designed to equip students with strong theoretical foundations and practical skills required in the modern IT industry.

Introduction to Programming

This course introduces fundamental programming concepts using a high-level programming language. Students learn problem solving, logic building, and basic coding practices.

Semester: 1st Semester

Data Structures and Algorithms

Covers essential data structures such as arrays, linked lists, stacks, queues, trees, and graphs along with algorithm design and complexity analysis.

Semester: 3rd Semester

Database Systems

Focuses on database design, SQL, normalization, and transaction management. Students gain hands-on experience with relational database systems.

Semester: 4th Semester

Software Engineering

Introduces software development life cycle models, requirement analysis, system design, testing techniques, and project management principles.

Semester: 5th Semester

Artificial Intelligence

Provides an overview of intelligent systems including search algorithms, knowledge representation, machine learning basics, and real-world AI applications.

Semester: 7th Semester

CS305-Software Engineering

Introduction to software development life cycle, agile models, and documentation.

Semester: Fall
